# Pac-Man Java Game

A desktop Pac-Man style game developed in Java using Swing and AWT.

## Features
- Playable Pac-Man character
- Random ghost movement
- Score tracking
- Lives system
- Win and Game Over screens
- Pause, Resume, and Restart buttons

## Technologies
- Java
- Swing
- AWT

## Project Structure
- `Game.java` - main game logic and application entry point
- `GamePanel.java` - main rendering panel and UI controls
- `Pacman.java` - player movement and score handling
- `Ghosts.java` - ghost movement and rendering
- `StartPanel.java` - start screen
- `images/` - game assets

## How to Run
1. Open the project in your Java IDE
2. Make sure the `images` folder is in the correct relative path
3. Run `Game.java`

## About the Project
This project was created as a Java game development exercise focused on object-oriented programming, event handling, rendering, and collision logic.
